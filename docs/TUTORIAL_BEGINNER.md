# Guia do Usuário — OmniMix

## O que é o OmniMix?

Mesa de controle virtual para transmissões ao vivo. Controla câmeras PTZ, vMix, ATEM e sequencias de cortes.

## Como abrir

1. Abra a pasta do OmniMix
2. Clique duplo em **`start.bat`**
3. O navegador abre automaticamente em http://localhost:5173

## Os dois modos

| Modo | Ícone | Para quê |
|---|---|---|
| Edição | ✏️ | Montar o painel, criar botões, conectar setas |
| Operacional | ▶️ | Usar durante a transmissão |

## Criando botões

1. Ative o Modo Edição
2. Clique em □ na toolbar → **Adicionar Botão**
3. Botão direito→ **Editar** para configurar o comando

## Setas (Workflow)

1. Ative as bolinhas (ícone Activity)
2. Arraste da bolinha direita → bolinha esquerda de outro botão
3. Uma seta aparece conectando os dois

## Sequenciador ⏭️

Lê todas as setas automaticamente:
- **NEXT** → executa o próximo passo da sequência
- **↺** → reinicia a sequência
- Clique num passo na lista para pular para ele

## Glossário

| Termo | Significado |
|---|---|
| Cut | Corte direto entre câmeras |
| Fade | Transição suave |
| PTZ | Câmera robótica |
| ATEM | Mesa de corte Blackmagic |
| PGM | O que está ao vivo agora |
| PVW | O que entra a seguir |
